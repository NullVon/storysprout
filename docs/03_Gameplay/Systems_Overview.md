# Systems Overview

StorySprout should eventually support the following systems.

## Core Systems

- Player movement
- Interaction prompts
- Dialogue boxes
- NPC conversations
- Inventory
- Quest items
- Companion/party following
- Memory discovery
- Turn-based encounters
- Mini-game framework
- Save/load

## Memory System

The player can discover optional moments.

Example:
- Memories Found: 17 / 24

For public projects, this can be framed as “Moments” or “Memories.”
For Playable Memory, it can directly support the theme of reliving meaningful memories.

## Encounter System

Turn-based encounters should support non-violent outcomes.

Examples:
- Compliment
- Gift
- Joke
- Talk
- Distract
- Help
