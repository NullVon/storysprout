# Game Design Document v0.3

## Core Concept

A cozy, top-down 2D RPG framework and game family centered on emotional storytelling, small handcrafted worlds, optional moments, and whimsical encounters.

## Design Philosophy

Not everything needs to be large or complex. The goal is to make small moments feel alive.

## StorySprout

StorySprout provides the reusable systems.

## First Bloom

First Bloom is the public/commercial cozy RPG built from StorySprout.

Working structure:
- Chapter 1: **Beginnings**
- Flower motif
- Cozy first-date-inspired adventure
- Generic, fictionalized story
- No proposal content

## Playable Memory

Playable Memory is the private proposal edition.

Structure:
- Act I: Anticipation
- Act II: Adventure
- Act III: Reflection and proposal

## Core Design Pillars

1. Cozy Exploration
2. Emotional Storytelling
3. Small Details Matter
4. Optional Wonder
5. No Punishment
6. Reusable Systems
