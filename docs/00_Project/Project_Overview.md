# Project Overview

## Studio Placeholder

**ZyCorps** is the placeholder studio identity for this learning project.

This is not being treated as a high-pressure company. It is a beginner-first, studio-style workflow meant to teach professional habits while building real games.

## Project Family

### 1. StorySprout

Public open-source framework.

Purpose:
- Learn professional software development.
- Build reusable cozy RPG systems.
- Create a portfolio-quality GitHub project.

### 2. First Bloom

Public/commercial cozy RPG.

Purpose:
- Build a publishable game using StorySprout.
- Keep the story generic and non-personal.
- Avoid any proposal or real relationship-specific content.

### 3. Playable Memory

Private personalized proposal edition.

Purpose:
- Create a custom game based on personal memories.
- Use the same systems proven in StorySprout and First Bloom.
- Include the proposal ending only in the private version.

## Public vs Private Rule

Anything public must avoid:
- Proposal content.
- Real names.
- Private relationship details.
- Direct personal callbacks such as “I heard wedding bells.”

The private Playable Memory project may include those details.
